# Formalidad e Informalidad Laboral por Sector en Chile

Este proyecto analiza la **formalidad e informalidad laboral por sector económico** en Chile, utilizando datos oficiales del **Instituto Nacional de Estadísticas (INE)**.

## Objetivo
Desarrollar un análisis exploratorio y visualización de la distribución del empleo formal e informal en los distintos sectores productivos, a partir de datos abiertos del INE.

## Fuente de datos
Los datos provienen del **INE** y están disponibles en su plataforma de estadísticas laborales:
- [Informalidad laboral - INE Chile](https://www.ine.gob.cl/estadisticas/sociales/mercado-laboral/informalidad-laboral/informalidad-y-condiciones-laborales)

## Estructura del repositorio
- `data/` → Contiene datasets originales y procesados.
- `notebooks/` → Contiene análisis exploratorios y visualizaciones.
- `scripts/` → Scripts de limpieza, transformación y carga de datos.
- `docs/` → Documentación técnica y reportes.

## Tecnologías
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Git & GitHub
